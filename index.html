Getting Started

MISSION

Supply a binary Linux OS that is easy to install and customize for both Desktop and Server.
LXD optimized distribution to permit the best ingration with lxd-compose tool.
Macaroni Linux OS is a young distribution under the Funtoo Linux umbrella and is developed and evolved primarily by Daniele Rondina (ex Sabayon Linux developer) aka geaaru.
The name Macaroni comes from the pinguin species found from the Subantarctic to the Antarctic Peninsula. It bears a distinctive yellow crest.
